# 🚁 Ground Control Station (GCS) for Swarm Applications

A browser-based **Ground Control Station (GCS)** built using **ROS**, **ROSBridge**, **MAVROS**, **HTML**, **CSS**, and **JavaScript** for controlling both single UAVs and drone swarms. The system provides an intuitive web interface for mission planning, real-time telemetry, swarm formation control, waypoint navigation, and autonomous area scanning. Designed with a modular architecture, it enables seamless communication between the browser and UAVs while remaining fully open-source, customizable, and capable of offline operation.

## ✨ Features

- 🚁 Single & Multi-UAV Control
- 📍 Interactive Waypoint Mission Planning
- 🛰️ Real-Time GPS Tracking
- 📊 Flight Telemetry Dashboard
- ⚙️ UAV Parameter Configuration
- 🎮 RC Calibration
- 👥 Swarm Formation Management
- 🗺️ Autonomous Area Scanning
- 🎯 Simultaneous Multi-UAV Command Execution
- 🌐 Browser-Based User Interface
- 📡 ROS & MAVROS Integration
- 💾 Offline Map Support
- 🔌 Easy Sensor Integration
- 📈 Flight Data Logging
